## 📖 About this Song
Song Tumi Bolechile Lyrics তুমি বলেছিলে ভালোবাসবে চিরকালহৃদয়ে রেখেছি তোমার সেই অঙ্গীকারকিন্তু আজ কেন এত শূন্যতাতোমার ছায়া খুঁজি আমি প্রতিবার রাতের আকাশে তারা হয়ে জ্বলেছিলেআমার জীবনে তুমি আলো ছড়িয়েছিলেপ্রতিটি মুহূর্ত সাজিয়েছিলে সুরেহঠাৎ কেন সব হারিয়ে গেল দূরে প্রতিশ্রুতি তোমার ভেসে যায়বাতাসেস্মৃতি শুধু বাঁচে মনেরআকাশেতোমাকে খুঁজি আমি প্রতিটিনিঃশ্বাসেকোথায় হারালে তুমিনিরুদ্দেশে তুমি বলেছিলে সাথে থাকবে চিরোকালতোমার কথায় […]
